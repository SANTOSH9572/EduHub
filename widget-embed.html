📋 Embed Code

Copy and paste this code into any website's HTML to add the study widget:

<script src="https://cdn.jsdelivr.net/gh/yourusername/eduhub@main/study-widget.js"></script>

🔗 Alternative CDN Links

If the main CDN doesn't work, try these alternatives:

<script src="https://raw.githubusercontent.com/yourusername/eduhub/main/study-widget.js"></script>
<script src="https://gitcdn.link/cdn/yourusername/eduhub/main/study-widget.js"></script>

💻 Local Installation

For local use, download the widget file and include it:

<script src="path/to/study-widget.js"></script>
